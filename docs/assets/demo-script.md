# Сценарий демо-GIF

1. Старт: показать полный HUD на тёмном фоне.
2. Навести на кнопки play / next / like, чтобы был виден neon hover.
3. Переключить HUD → Slim.
4. Переключить Slim → Orb.
5. Правый клик по Orb → открыть Settings.
6. Показать вкладку Effects: Low / Normal / Aggressive.
7. Закрыть окно → показать, что приложение остаётся в tray.

Рекомендации:

- длительность: 10–15 секунд;
- размер: 1280x720;
- FPS: 30;
- без лишнего рабочего стола;
- лучше записывать на тёмном фоне.
