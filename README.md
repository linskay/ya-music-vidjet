# Ya Music Vidjet

Мини-виджет для Яндекс Музыки на Windows: киберпанк UI, управление воспроизведением и подготовка к сборке в `.exe`.

## Идея

Приложение показывает маленький плавающий виджет поверх окон и управляет Яндекс Музыкой через встроенный WebView. Цель — не переписывать музыкальный сервис и не обходить API, а аккуратно спрятать тяжёлый интерфейс сайта и оставить только нужные кнопки.

## MVP

- Java 21 + JavaFX.
- Плавающее окно без стандартной рамки.
- Cyberpunk neon UI: контурные кнопки, тёмная стеклянная панель, cyan/magenta glow.
- Управление: play/pause, previous, next, like, dislike, volume.
- Кнопка `Моя волна`.
- Скрытый/служебный WebView для `https://music.yandex.ru/`.
- JS-инъекции для кликов по элементам страницы Яндекс Музыки.
- Упаковка Windows `.exe` через `jpackage`.

## Быстрый старт

```bash
./gradlew run
```

Windows:

```powershell
.\gradlew.bat run
```

## Сборка приложения

```bash
./gradlew clean jpackage
```

Результат будет в:

```text
build/jpackage/
```

Для создания installer `.exe` на Windows потребуется WiX Toolset, если используется режим MSI/EXE installer. Runtime image можно собрать без WiX.

## Статус

Это стартовый skeleton проекта. UI уже отделён от слоя управления плеером, чтобы потом заменить заглушки на реальные JS-команды WebView.
